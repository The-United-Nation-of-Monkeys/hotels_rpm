# Быстрый старт

## 1. Активация виртуального окружения
```bash
source venv/bin/activate
```

## 2. Применение миграций (уже выполнено)
```bash
python manage.py migrate
```

## 3. Создание суперпользователя для админ-панели
```bash
python manage.py createsuperuser
```
Введите имя пользователя, email и пароль.

## 4. Запуск сервера
```bash
python manage.py runserver
```

## 5. Откройте в браузере
- **Главная страница**: http://127.0.0.1:8000/
- **Админ-панель**: http://127.0.0.1:8000/admin/

## Что дальше?

1. Войдите в админ-панель и создайте несколько номеров отеля
2. Создайте несколько гостей через админ-панель или через веб-интерфейс
3. Создайте бронирования через веб-интерфейс

## Основные страницы

- `/` - Главная страница
- `/rooms/` - Список номеров
- `/guests/` - Список гостей
- `/bookings/` - Список бронирований
- `/contact/` - Контакты
- `/admin/` - Админ-панель
